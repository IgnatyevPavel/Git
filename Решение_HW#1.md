#### Как выполнять задание?
>Для выполнения задания у вас должен быть установлен для Windows - GitBash.
>Создан аккаунт в GitHub
>Все шаги сценария выполняйте в терминале GitBash, Terminal, в папке под гитом.
>Как отправить ДЗ на проверку.
> 1. Создайте текстоовый файл как в первом ДЗ по Terminal.
> 2. Сценарий перенесите в этот файл.
> 3. На против каждого действия - напишите команду в GitBash
>
>Файл со сценарием и ссылку на свой гит хаб отправляйте менторам на проверку.


### JSON
*4. Создать внешний репозиторий c названием JSON.*
+ На главной странице профиля в `GitHub` во вкладке `Repositories` кликните на кнопку `New` 
+ В открывшейся странице заполните форму `Create a new repository`  
    + Заполните поле `Repository name` впишите имя: `JSON`
    + Поставьте галочку на `Public` и `Add readme file` 
+ Нажмите `Create repository.` 
___
*5. Клонировать репозиторий JSON на локальный компьютер.*
+ На главной странице профиля в `GitHub` во вкладке `Repositories` нажмите на созданный репозиторий `JSON` 
+ Кликните на кнопку `Code` 
+ Во вкладке `HTTPS` скопируйте ссылку на репозиторий
+ Откройте окно терминала, выберите директорию куда сохранить репозиторий
+ Впишите в терминал команду: 
```git
git clone https://github.com/IgnatyevPavel/JSON.git
```
___
*6. Внутри локального JSON создать файл “new.json”.*
+ Перейдите в локальный репозиторий `JSON`: 
```git
cd JSON 
```
+ Создайте файл `new.json`:
```git
touch new.json
```

___
*7. Добавить файл под гит.*
+ Чтобы убедиться что созданный файл `new.json` в статусе `untracked` введите команду:
```git
git status 
```
+ Добавьте файл с помощью комады:
```git
git add new.json
```
+ Проверьте изменённый статус файла:
```git
git status 
```
___
*8. Закоммитить файл.*
```
git commit -m "file 'new.json' has been created"
```
___
*9. Отправить файл на внешний GitHub репозиторий.*
```
git push
```

*10. Отредактировать содержание файла “new.json” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате JSON.*
